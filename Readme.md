# A/B Testing Analysis for Landing Page Optimization

## 📌 Overview
This project analyzes an A/B test to determine whether a new landing page improves conversion rates.

## 🎯 Objective
Evaluate if the treatment group leads to higher conversions than the control group.

## 📊 Dataset
Real-world A/B testing dataset (Kaggle)

## ⚙️ Methodology
- Data cleaning (removed mismatches & duplicates)
- Conversion rate analysis
- Uplift calculation
- Two-proportion Z-test
- Confidence intervals
